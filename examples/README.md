# examples/

There's no bundled sample corpus. This folder is an optional fallback location for a shared/demo corpus — the lab reads it via `SAMPLE_CORPUS_PATH` only when `knowledge_base/` has no `meta.yaml` files.

For normal use, add your own documents under `knowledge_base/` instead — see [`knowledge_base/README.md`](../knowledge_base/README.md) for the format, and [`STUDENT_GUIDE.md`](../STUDENT_GUIDE.md) step 3 for a full walkthrough.
